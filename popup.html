OSM iD Session Time Tracker for ROVAS

This extension tracks the time spent working on OpenStreetMap iD and automatically submits work reports to the ROVAS App.

NOTE: To use this extension, you must be a registered user in Rovas and have your API KEY and TOKEN (available on your account page). You also need to be a shareholder of the OpenStreetMap project (ID 1998).
